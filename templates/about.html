About

Mustang Map provides a visualization of data provided by the Bureau of Land Management's Mustang Program. The BLM provides this data in the form of pdf - sometimes scans of printed pdfs. Mustang Map's database is based on the provided data, but may have small errors due to potential pdf to text conversion issues. This site is intended for visualization of data and should not be considered 100% accurate.

Pictures

All pictures on this site are provided by users who have certified that they have the right to distribute these pictures. If you have found a picture that belongs to you and want it removed, please email Mustang Map.

RESTful API

The data visualized in Mustang Map is available as a RESTful API.
For information on using this API, please see the documentation here.

Footnotes

In cases where the data presented by the Bureau of Land Management was blank and a numerical value was necessary for charting, that data is presented as 0. A footnote in the charts indicates this substitution. To see the original reported data, please visit the BLM's Mustang Program.
